# 截图资源说明

此目录用于存放 README 中引用的截图。请按照以下说明添加截图：

## 需要的截图

### 1. notion_integration.png

**内容**：Notion Integration 创建页面
**说明**：展示如何创建 Integration 并复制 Token

### 2. add_connection.png

**内容**：Notion 页面添加 Integration
**说明**：展示在页面设置中添加 Integration 连接

### 3. cookiecloud_config.png

**内容**：CookieCloud 浏览器插件配置界面
**说明**：展示插件的服务器地址、UUID、密码配置

### 4. github_secrets.png

**内容**：Github Secrets 配置页面
**说明**：展示添加 CC_URL、CC_ID、CC_PASSWORD 等 Secrets

### 5. manual_cookie.png

**内容**：浏览器开发者工具中复制 Cookie
**说明**：展示在 Application → Cookies 中查看 weread.qq.com 的 Cookie

### 6. run_workflow.png

**内容**：Github Actions 手动触发界面
**说明**：展示点击 Run workflow 按钮触发同步

## 截图建议

- 使用 1280x720 或更高分辨率
- 对敏感信息（如 Token、密码）进行打码处理
- 可使用 Windows 截图工具 (Win+Shift+S) 或 Mac 截图 (Cmd+Shift+4)
- 保存为 PNG 格式以保证清晰度
